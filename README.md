# 👚👕👖옷장을 부탁해 🛍️👛👞 


## 💻 프로젝트 개요

**👕 날씨, 취향을 고려해 사용자가 보유한 의상 조합을 추천해주고, OOTD 피드, 팔로우 등의 소셜 기능을 갖춘 서비스**

> 코드잇 스프린트 고급 프로젝트 - 옷장을 부탁해 
> 개인화 의상 및 아이템 추천 SaaS 서비스 SPRING 백엔드 구축  
> 개발 기간: 2025.06.23 ~ 2025.07.31
